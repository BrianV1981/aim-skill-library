# aim-memory-search (aim-grok)

From your vessel project root:

```bash
./aim map
./aim search "<query>"
./aim search "<query>" --top-k 10
./aim search "<snippet>" --full
./aim wiki search "<query>"
```

Prefer `map` before open-ended exploration. If nothing is found, say so — do not invent Engram facts.
