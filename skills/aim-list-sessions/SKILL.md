# aim-list-sessions

From the vessel project root:

```bash
./aim map
./aim doctor
```

Useful companions:

```bash
git log --oneline -20
# Grok CLI transcripts (if using Grok): ~/.grok/sessions/
```

Report a short inventory: what memory/index exists, health notes from `doctor`, recent code history if relevant.
